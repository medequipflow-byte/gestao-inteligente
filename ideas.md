## Abordagem 1: Elegância Clínica Minimalista
<response>
<text>

**Core Principles**:

- Dados e clareza como estética

**Interaction Philosophy**: Hover suave com elevação de cards; botões com transição de cor fluida

**Animation**: Entrada de elementos com fade-up escalonado; carrossel com deslize suave

## Abordagem 3: Orgânico-Premium (ESCOLHIDA)
<response>
<text>
**Design Movement**: Soft Luxury Health — wellness meets enterprise
**Core Principles**:
- Gradientes suaves como linguagem visual
- Cantos arredondados generosos para acolhimento

**Color Philosophy**: Creme (#FDFAF5) como fundo base cria sensação de papel premium; lavanda (#B8A9E8 / oklch) como cor de destaque evoca inovação e cuidado; azul meia-noite (#1B2B5E) para texto e elementos de autoridade.

**Animation**: Fade-up escalonado na entrada; carrossel com easing spring

**Typography System**: Cormorant Garamond (títulos display, peso 600) + Nunito Sans (corpo, peso 400/600)
